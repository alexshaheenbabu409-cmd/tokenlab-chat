# TokenLab Chat Engine 🚀

একটি অত্যন্ত শক্তিশালী, আধুনিক এবং মেমোরি-সমৃদ্ধ Node.js চ্যাট ব্যাকএন্ড ইঞ্জিন। এটি অফিশিয়াল `openai` SDK ব্যবহার করে কাস্টম এপিআই গেটওয়ে (যেমন: TokenLab API) এবং লাইভ ইন্টারনেট সার্চ টুলের সাথে সংযুক্ত হয়ে কাজ করে। প্রজেক্টটি কোনো ডকার ফাইল ছাড়াই সরাসরি **Sevalla**, Vercel বা যেকোনো Node.js এনভায়রনমেন্টে ১-ক্লিকে ডেপ্লয় করার জন্য উপযোগী করে ডিজাইন করা হয়েছে।

## 🌟 ফিচারসমূহ (Features)

### ১. অফিশিয়াল OpenAI SDK v4 আর্কিটেকচার
সার্ভারটি ম্যানুয়াল HTTP রিকোয়েস্টের পরিবর্তে অফিশিয়াল `openai` নোড প্যাকেজ ব্যবহার করে। এর ফলে ব্যাকএন্ড কোড অত্যন্ত স্টেবল, ক্লিন এবং দীর্ঘমেয়াদী ব্যবহারের জন্য নিরাপদ।

### ২. ডায়নামিক মডেল সিলেকশন (Dynamic Model Flexibility)
* **ডিফল্ট মডেল:** কোনো মডেল উল্লেখ না থাকলে সার্ভার স্বয়ংক্রিয়ভাবে সুপার-ফাস্ট `deepseek-v4-flash` ব্যবহার করবে।
* **মাল্টি-মডেল স্বাধীনতা:** ক্লায়েন্ট বা ফ্রন্টঅ্যান্ড থেকে রিকোয়েস্ট বডিতে (`model`) যেকোনো OpenAI স্ট্যান্ডার্ড সমর্থিত মডেলের নাম (যেমন: `gpt-4o`, `claude-3-5-sonnet`) পাঠিয়ে সরাসরি চ্যাট করা সম্ভব। কোড পরিবর্তন করার কোনো প্রয়োজন নেই।

### ৩. লাইভ ইন্টারনেট সার্চ (Tavily AI Integration)
ইউজারের মেসেজে সাম্প্রতিক কোনো জিজ্ঞাসা (যেমন: "আজকের খবর", "এখনকার আবহাওয়া", "সাম্প্রতিক লাইভ স্কোর" ইত্যাদি) থাকলে ব্যাকএন্ডের স্মার্ট কিওয়ার্ড লজিক তা নিজে থেকেই শনাক্ত করে। এরপর **Tavily Search API** ব্যবহার করে রিয়েল-টাইম ইন্টারনেট ডেটা নিয়ে মূল প্রম্পটের সাথে যুক্ত করে মডেলের কাছে পাঠায়।

### ৪. ইন-মেমোরি সেশন ও চ্যাট হিস্ট্রি (Smart Session Memory)
প্রতিটি ইউজারের জন্য আলাদা `user_id` ভিত্তিক ইন-মেমোরি চ্যাট মেমোরি ম্যানেজমেন্ট। এআই পূর্বের কথোপকথনের প্রসঙ্গ ও ইতিহাস মনে রেখে বুদ্ধিদীপ্ত উত্তর দিতে পারে। ডিফেন্ডেন্সি হিসেবে সর্বোচ্চ ২০টি মেসেজ ব্যাকলগ ট্র্যাকিং লজিক যুক্ত আছে।

### ৫. কাস্টম সিস্টেম প্রম্পট (System Prompt Injection)
পরিবেশের ভ্যারিয়েবল (`SYSTEM_PROMPT`) ব্যবহার করে এআই-এর আচরণ, ব্যক্তিত্ব (যেমন: নিজেকে "নবিতা" বা নির্দিষ্ট কাস্টম অ্যাসিস্ট্যান্ট হিসেবে পরিচয় দেওয়া) এবং উত্তর দেওয়ার ধরন পুরোপুরি নিয়ন্ত্রণ করা যায়।

## 🛠️ এপিআই এন্ডপয়েন্ট (API Endpoints)

### ১. সার্ভার স্ট্যাটাস চেক (Root)
* **Method:** `GET`
* **Path:** `/`
* **কাজ:** সার্ভার অনলাইনে সফলভাবে সেশন বুট করেছে কি না তা পরীক্ষা করা।

### ২. মূল চ্যাট ইঞ্জিন
* **Method:** `POST`
* **Path:** `/chat`
* **Request Body Example:**
```json
{
  "user_id": "shahin_user_01",
  "message": "আজকের সাম্প্রতিক আন্তর্জাতিক খবরগুলো কী কী?",
  "model": "deepseek-v4-flash"
}
